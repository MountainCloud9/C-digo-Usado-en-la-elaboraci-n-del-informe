# Resultados

Esta carpeta contiene los resultados generados por los scripts de análisis ubicados en `scripts/analisis/`.

## Descripción

Aquí se almacenan tablas, archivos de datos de salida, modelos entrenados y cualquier otro resultado producido durante el análisis.

## Notas

- Esta carpeta puede estar vacía si los resultados son demasiado grandes o si deben generarse ejecutando los scripts.
- Para reproducir los resultados, sigue las instrucciones en `scripts/analisis/README.md`.
