## Fórmula General

Si tu concentración madre es **C** mg/mL, calcula para cada punto:

```
Volumen de solución madre (uL) = (Concentración deseada × 1000) / C

Volumen de etanol (uL) = 1000 - Volumen de solución madre
```

**Ejemplo:** Para concentración deseada 0.55 mg/mL con C = 3 mg/mL:
```
V = (0.55 × 1000) / 3 = 183.33 uL
```

## Notas Importantes

1. Los volúmenes mostrados son para pipetear de la **solución madre**
2. Completar siempre con **etanol puro** hasta **1000 uL total**
3. Si tu concentración madre no está en la lista, usa la fórmula general
4. Los volúmenes están redondeados a 1 decimal (ej: 36.667 → 36.7)
5. Para pipeteo más exacto, usa decimales completos

---

## Verificación Rápida

Puedes verificar que un cálculo es correcto:
```
(Volumen solución madre × Concentración madre) / Volumen total = Concentración final

Ejemplo: (36.667 × 3) / 1000 = 110 / 1000 = 0.11 mg/mL ✓
```
