# Análisis

Scripts para el análisis estadístico y modelado de los datos procesados.

## Descripción

Coloca aquí los scripts de análisis principal: estadísticas descriptivas, pruebas de hipótesis, modelos, etc. Los scripts deben leer de `datos/procesados/` y guardar resultados en `resultados/`.

## Orden de ejecución

Ejecuta los scripts en el orden numérico o alfabético indicado en sus nombres (e.g., `01_estadisticas.py`, `02_modelo.py`).
