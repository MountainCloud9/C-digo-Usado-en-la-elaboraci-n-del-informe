# Visualización

Scripts para la generación de gráficas y figuras del proyecto.

## Descripción

Coloca aquí los scripts que producen las gráficas e imágenes incluidas en el informe. Los scripts deben leer de `datos/procesados/` o `resultados/` y guardar las figuras en `figuras/`.

## Orden de ejecución

Ejecuta los scripts en el orden numérico o alfabético indicado en sus nombres (e.g., `01_grafica_distribucion.py`, `02_grafica_comparacion.py`).
