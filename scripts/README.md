# Scripts

Esta carpeta contiene todos los scripts utilizados para el procesamiento y análisis de datos.

## Subcarpetas

- **preprocesamiento/**: Scripts para la limpieza, transformación y preparación de los datos brutos.
- **analisis/**: Scripts para el análisis estadístico y modelado.
- **visualizacion/**: Scripts para la generación de gráficas y figuras.

## Cómo ejecutar

Consulta el archivo `README.md` dentro de cada subcarpeta para instrucciones específicas sobre dependencias y orden de ejecución.
