# Preprocesamiento

Scripts para la limpieza y preparación de los datos brutos ubicados en `datos/brutos/`.

## Descripción

Coloca aquí los scripts que transforman los datos brutos en datos procesados. Los scripts deben leer de `datos/brutos/` y escribir en `datos/procesados/`.

## Orden de ejecución

Ejecuta los scripts en el orden numérico o alfabético indicado en sus nombres (e.g., `01_limpieza.py`, `02_transformacion.py`).
