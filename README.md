# Código Usado en la Elaboración del Informe

Repositorio de transparencia que contiene todo el código utilizado durante el proyecto de investigación, desde el anteproyecto hasta el informe final.

## Estructura del Repositorio

```
├── datos/              # Conjuntos de datos utilizados en el análisis
│   ├── brutos/         # Datos originales sin procesar
│   └── procesados/     # Datos limpios y transformados
├── scripts/            # Scripts de análisis y procesamiento
│   ├── preprocesamiento/   # Scripts de limpieza y preparación de datos
│   ├── analisis/           # Scripts de análisis estadístico
│   └── visualizacion/      # Scripts de generación de gráficas
├── resultados/         # Resultados generados por los scripts
└── figuras/            # Gráficas e imágenes generadas
```

## Descripción

Este repositorio almacena de forma transparente todos los scripts, análisis y procedimientos computacionales empleados en la elaboración del anteproyecto e informe final de investigación.

Cada carpeta contiene un archivo `README.md` con una descripción detallada de su contenido y cómo ejecutar los scripts correspondientes.

## Requisitos

Consulta el archivo `README.md` dentro de cada carpeta de `scripts/` para conocer las dependencias específicas de cada análisis.

## Reproducibilidad

Para reproducir los resultados:

1. Clona este repositorio.
2. Instala las dependencias indicadas en cada subcarpeta.
3. Ejecuta los scripts en el orden descrito en cada `README.md`.
